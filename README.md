# Top Global Universities

An interactive web application showcasing the top 30 universities worldwide. Browse through university rankings, explore detailed information about each institution, and discover their signature programs and locations.

## What This Is

A clean, fast-loading website that displays curated information about the world's leading universities. Features a split-screen interface: select a university from the list to view its full details, including location, description, and top academic programs.

Optimized for iPad viewing but works on all modern devices.

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the site.

## Build for Production

```bash
npm run build
npm run start
```
